# 了解NEO
## 学习目标
了解NEO，了解NEOGUI的基本使用，了解获取GAS的方法，了解转账方法

## 需要

1.懂得C#编程

2.懂得GIT的使用

## 步骤

1.准备windows 10电脑一台，更新到最新。

2.安装vs2017，并安装dotnet 4.7开发环境，和 dotnet core2.0 开发环境。

安装dotnet4.7开发环境的验证方法为可以用vs2017 创建 dotnet4.7的项目（注意dotnet 4.7.1 开发环境 并不包含 dotnet4.7 开发环境）

安装dotnet core2.0 开发环境的验证方法为 可以用命令行 输入 dotnet --version 看到2.0.x的版本号，并且可以使用 dotnet new 指令创建新的dotnet core 2.0 项目

3.看官方资料了解NEO

官网(http://neo.org)

4.获取NEOGUI项目源码并生成

如果NEO官方项目生成困难，我们无意解答这些问题。

可换用NEL定制版本，生成比较简单(https://github.com/NewEconoLab/neo-gui-nel)

5.搞清楚 NEO的 MainNet 和 TestNet 两条链，确保NeoGUI在TestNet 链上更新

6.了解NeoGUI的基本使用

7.通过各种手段获取TestNet上的Gas

如果获取GAS碰到麻烦，可到NEL交流群里面索取

8.进行GAS转账测试

完成学习
